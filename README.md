# Ribatul Maulid App

Aplikasi Android berbasis Kotlin yang menyediakan informasi keislaman, artikel, berita, jadwal sholat, dan konten edukasi Islam. Aplikasi ini dikembangkan menggunakan Android Studio sebagai tugas proyek kampus.

## 📱 Fitur

- Beranda artikel dan berita Islami
- Detail artikel
- Kategori artikel
- Jadwal sholat
- Profil aplikasi
- Pencarian artikel
- Tampilan modern dan responsif

## 🛠️ Teknologi

- Android Studio
- Kotlin
- PHP REST API
- MySQL Database
- RecyclerView
- Glide
- Retrofit

## 📂 Struktur Project

```
app/
├── activities/
├── adapters/
├── api/
├── models/
├── fragments/
├── utils/
└── res/
```

## 🚀 Instalasi

1. Clone repository

```bash
git clone https://github.com/yazidsyafrudin/ribatul-maulid-app.git
```

2. Buka project di Android Studio

3. Sync Gradle

4. Jalankan aplikasi pada emulator atau perangkat Android
